# Class-6
El juego funciona de la siguiente manera:

    1.- Se le da la bienvenida al jugador pidiendole que adivine un numero entre el 1 y el 100, en 7 intentos.

    2.- El jugador tiene que ingresar un numero valido (entero y dentro del rango).

    3.- Si el numero dado por el jugador es incorrecto se le dara uno de dos mensajes:
        "Tu numero es muy alto." si el numero es mas grande que el que es correcto
        "Tu numero es muy bajo." si el numero es mas pequeño que el que es correcto.

    4.- Si el numero dado es igual al correcto te dara el mensaje:
        "¡Felicidades! Adivinastes en x intentos."
        en este caso x es el numero de intento en el cual se adivino el numero.

    5.- Si el jugador no adivina el numero en todos sus intentos se le dara el mensaje:
        "¡Se acabaron los intentos! El numero secreto era x."
        en este caso x siendo el numero correcto.

    6.- Al final del juego se te dara la opcion de jugar de nuevo, si quieres jugar de nuevo escribe "s", si no escribe "n"
